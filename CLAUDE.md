# Лев и Валера — память проекта

Раннер-игра для сына пользователя **Льва (6 лет)** и его воображаемого друга **Валеры** (красный мармеладный мишка-Haribo). Играют на телефоне, в ландшафте.

- **Репозиторий:** https://github.com/asaenokkostya-coder/lev-and-valera

## Про Льва (визуал персонажа)

Рисуется процедурно на Canvas в `drawLev()`. При правках внешности — менять там:
- Кожа: `#FFD5B8` (персиковая)
- Волосы: сверху торчащие спики `#B5A07A` (русый с оттенком золота), бока буз `#8B7556` (темнее)
- Глаза: бело-голубо-серые, `#6B8CA5`
- Футболка: тёпло-серая с коричневым `#9D8277` (как на фото)
- Улыбка и намёк на зубы (ребёнок меняет зубы — отсюда и «Охотник за кариесом»)

## Про Валеру (визуал персонажа)

Рисуется в `drawValera()`:
- Основной красный: `#E52B43`, тень: `#C11938`, блик: полупрозрачный белый
- Мордочка и животик: светло-розовые `#F06A7D`
- Тёмные глазки/носик: `#2C0810`
- Бежит рядом со Львом, чуть позади и ниже по скорости анимации — чтобы было видно что это «хвостик»

## Техническая архитектура

- **Рендер:** Canvas 2D (не SVG — быстрее для скролла)
- **Виртуальная резолюция:** 900×500, масштабируется с letterboxing под любой экран
- **Физика:** простая гравитация (`GRAVITY = 2000`), прыжок (`JUMP_VEL = -820`)
- **Скроллинг:** мир движется влево, Лев зафиксирован на `PLAYER_X = 220`
- **Игровой цикл:** requestAnimationFrame с clamped dt
- **Хранение рекорда:** localStorage ключ `lev-high`
- **Звук:** Web Audio API, осцилляторы — без внешних файлов
- **Ориентация:** принудительно ландшафт (портрет на телефоне показывает заглушку «Поверни телефон»)

## Геймплей

- Авто-бег слева направо, тап/Space = прыжок (только когда на земле)
- **3 жизни**, неуязвимость 1.4 сек после удара
- **Единственный хазард — лужи фиолетового мёда** на земле. Прыгать через.
- **Оса сзади (chaser)** — одна, постоянно висит на левом краю экрана, слегка покачивается. Когда Лев проваливается в мёд — оса резко пикирует к нему, жалит (−1 жизнь), возвращается на место. Это даёт ощущение погони, но требует следить только за одним: землёй.
- **Звёзды** — собирать, +15 очков бонусом
- **Скорость растёт** с временем (до потолка 580 px/s)
- **Score** = расстояние / 10 + бонусы за звёзды
- **Milestone popups** каждые 100 очков

## Константы сложности (крутить здесь)

В начале `<script>`:
- `SPEED_START` (300) — стартовая скорость
- `SPEED_MAX` (580) — потолок
- `SPEED_ACCEL` (6) — насколько быстро разгоняется
- `JUMP_VEL` (-820), `GRAVITY` (2000) — ощущение прыжка
- Тайминги спавна: `spawnHoney`, `spawnWasp`, `spawnStar` в `resetGame()`

### 2026-04-23 — v4 «Mario / Rayman фишки»
- Обратная связь после v3: «прикольно, можешь добавить фишки из Mario/Rayman?»
- **Пружины-батуты** 🍄: красный мухомор на ножке-пружине (как Mario grow-top). Массив `world.springs`, спавн каждые 7-12 сек. При соприкосновении: `player.vy = JUMP_VEL * 1.65` (в ~1.65 раза выше обычного прыжка), сбрасывает `canDoubleJump`. Анимация сжатия (`bouncePhase`). Popup «БУМ!». Новый звук `sfx.spring`.
- **Радужная Супер-звезда** 🌈 (Mario's Super Star): массив `world.superStars`, спавн каждые 18-28 сек, редкая и ценная. При сборе даёт `player.superMode = 5.0` сек. В супер-режиме: Лев окружён радужной аурой (`drawSuperAura`), пробегает через мёд без урона (splash-эффект), двойной прыжок становится **бесконечным** (пока действует), частицы за ним цветные. Новый звук `sfx.super` — восходящая мажорная арпеджио.
- **Арки звёзд** (Rayman lum chains): в `spawnStar` с шансом 25% спавнится не одна звезда, а четыре в форме дуги — стимулирует точные двойные прыжки.
- Новая графика: `drawSpring`, `drawSuperStar` (с радужными обводками и смайликом), `drawSuperAura`.

### 2026-04-23 — v3 «добавить изюма»
- Обратная связь: «скучновато»
- **Двойной прыжок:** тап в воздухе = второй прыжок (82% силы первого). Флаги `player.canDoubleJump`, `doubleJumped`. Визуально — золотые искры. Валера тоже дублирует.
- **Серии (combo):** поле `world.combo` увеличивается при прохождении мёда, сбрасывается на 0 при ударе. Тайлы ×2 с 3-х, ×3 с 6-ти, ×5 с 10-ти. Умножает очки за звёзды. Отображается в HUD по центру сверху (пульсирует), popup «СЕРИЯ ×N!» при пересечении порога.
- **Биомы:** массив `BIOMES[4]` (утро / день / закат / ночь), выбор через `getBiome(score)` по порогам 200 / 500 / 900. Меняются небо (градиент), солнце/луна, земля, трава, грязь. В ночном биоме добавлены мерцающие звёзды.
- **Вариативность мёда:** `spawnHoney` теперь роллит 3 категории: узкая (55-85px, 50%), средняя (95-130px, 35%), широкая (150-200px, 15%).
- **`bestCombo`** сохраняется в `world`, показывается на экране Game Over.
